### Customization Tips

You can easily customize the content by editing the HTML file:

- **Love Letter**: Modify the text in the `.love-letter` section (around line 250)
- **Appreciation Cards**: Update the three `.card` elements with your own messages
- **Surprise Messages**: Edit the `messages` array in the JavaScript (around line 560)
- **Colors**: Change CSS variables in the `:root` section for different color schemes
- **Fonts**: Replace Google Fonts links to use different typography
